Here you can store cards. 
Each card asset should have an EXACT size of 550 × 382 pixels and be saved in png format.
The left side should be the IMAGE, while the right side should be the DESCRIPTION (spells, hp, etc).

==============================================

Répertoire pour les cartes.
Chaque carte doit avoir une taille exacte de 550*382 pixels et être sauvée en format png.
La partie de gauche est l'IMAGE, et celle de droite la DESCRIPTION (attaques, vie, etc).
